# HakedisApp 🏗️

İnşaat sektöründe kullanılan hakediş (progress payment) süreçlerini yönetmek için geliştirilmiş iOS uygulaması.

## Özellikler

- **Proje Yönetimi** — Birden fazla proje, blok/kat/mahal yapısı
- **Taşeron Takibi** — Sözleşmeler, alacak/ödeme durumu
- **Günlük Saha Girişi** — Hızlı metraj kaydı, fotoğraf ekleme
- **Otomatik Hakediş** — Dönem seçince sistem hesaplar, kümülatif takip
- **Onay Süreci** — Taslak → Onay → Ödendi akışı
- **Ödeme Takibi** — Kısmi ödeme, gecikme görünümü
- **Raporlar** — Proje ve taşeron bazlı finansal özet

## Teknoloji

- SwiftUI
- SwiftData
- iOS 17+

## Kurulum

`setup_instructions.md` dosyasını takip edin.

## Uygulama Akışı

```
Şantiye → Günlük Giriş → Metraj → Hakediş → Onay → Ödeme
```
